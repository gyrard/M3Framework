Towards the Linked Open Reasoning.
This web site is dedicated to semantics-based reasoning mechanisms applied to Internet of Things (IoT), Web of Things (WoT) and smart cities.

To correctly visualize this web site, please consider using a browser such as Chrome or Firefox for a better result and being able to run javascript. Thank you.
