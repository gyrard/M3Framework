  • M3 language and its semantic annotator documentation (PDF):
    This documentation explains our M3 language to describe unify sensor data.
    • The M3 language can be considered as a nomenclature or a dictionnary or a taxonomy to unify standardize sensor metadata (sensor type, measurement type, unit and domains)sensor data. By following this language, it will be easier to automate the reasoning.

    • The M3 language is implemented within the M3 ontology and the updated version M3-lite taxonomy (V2).

    • The semantic annotator assists users in semantically annotating data to unify sensor data and being compliant with the M3 language.

  • Paper: Standardizing Generic Cross-Domain Applications in Internet of Things [Gyrard et al. Globecom Workshop 2014]
