M3-lite taxonomy (V2)

Tool/Demo		 This documentation explains our M3 Nomenclature (= dictionary = language) to describe sensor metadata in a unified/standardized way (sensor type, measurement type, unit and domains). By following this nomenclature to describe sensor metadata, it will be easier to automate the reasoning.
The M3 nomenclature is implemented within the M3 ontology and the updated version M3-lite taxonomy.
The semantic annotator assists in semantically annotating data and being compliant with the M3 language (dictionary = nomenclature). 		June 2016
