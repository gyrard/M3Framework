Machine-to-Machine Measurement (M3) is a framework to semantically annotate and easily interpret Internet of Things (IoT) data.
M3 enables designing interoperable domain-specific or cross-domain Semantic Web of Things (SWoT) applications.
M3 comprises the following components:
Generating semantic-based IoT applications
Reusing domain knowledge
Semantic Web
Best Practices
Interpreting IoT data
Interoperable IoT Data
& Domain Knowledge
Securing IoT applications
